The restrictions of the Jefferson Parish Comprehensive Zoning Ordinance shall not be deemed to be modified by any provision of this Code. When in conflict, the Comprehensive Zoning Ordinance restrictions shall control, except where this Code imposes greater restrictions by construction type and/or occupancy use then the provisions of this Code shall govern.
