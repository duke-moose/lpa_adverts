In accordance with Section 1.01(16) of the Jefferson Parish Charter, the Louisiana State Uniform Construction Code, La. R.S. 40:1730.21 et seq. as amended, comprised of the following codes, less and except Parts I—Administrative thereof, are hereby adopted as the Building Code of the Parish of Jefferson by reference as though they were copied here fully and are applicable in the unincorporated area of Jefferson Parish:

International Residential Code for One- and Two-Family Dwellings
International Fuel Gas Code
International Mechanical Code
International Plumbing Code
National Electrical Code
International Existing Building Code
