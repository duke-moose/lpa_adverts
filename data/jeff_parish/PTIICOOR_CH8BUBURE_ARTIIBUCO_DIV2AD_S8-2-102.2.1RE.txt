Whenever necessary to make an inspection to enforce any provisions of this chapter, or whenever the code official has reasonable cause to believe that there exists in any building or upon any premises any condition or code violation which makes such building, structure, premises, electrical, gas, mechanical or plumbing systems unsafe, dangerous or hazardous, the code official is hereby authorized and empowered to enter such building, structure or premises at all reasonable times to inspect same or to perform any duty imposed upon the code official by this chapter, provided that such building or premises are occupied. The code official shall first present proper credentials and request entry. If such building, structure, or premises are unoccupied, the code official shall first make a reasonable effort to locate the owner or other persons having charge or control of such and request entry. If entry is refused, the code official shall have recourse to every remedy provided by law to secure entry.
