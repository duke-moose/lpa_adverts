The code official shall act upon an application for a permit without unreasonable or unnecessary delay. If the code official is satisfied that the work described in an application for a permit and the contract documents filed therewith conform to the requirements of the technical codes, the comprehensive zoning ordinance, and other pertinent laws and ordinances, the code official shall issue a permit to the applicant for the specified construction and/or installation. All state-mandated agency approvals shall be received before the code official shall issue any building permit.
